Squad Command System

BT + Input System으로 아군 캐릭터 이동/공격/정지/집합/비살상/스킬/아이템/상호작용 8가지 명령 통제. Command 추상화로 BT가 실행·전환 담당. BlackBoard에 약 30개 데이터 캐싱으로 중복 연산 제거.

Behavior Tree

FSM 대신 BT 채택. 사수/탱커/힐러/채광 4개 특성별 BT를 캐릭터 생성 시 초기화. 재사용 가능한 Node 분리. 적 AI Idle → Warning → Chase → Attack 단계 구현.

Game System

Skill (SO + Factory), Noise (스프레드/에어리어 × 싱글/멀티 × 인스턴트/서스테인/에코), 인벤토리, 미션, 카메라, VFX, Sound Manager, Buff 시스템 총 10종 구현.

최적화

BT tick 분산(0.1~0.2s), OverlapSphereNonAlloc, SqrMagnitude 적용. 시야 시스템 GC Alloc 200KB+ → 0B 수준으로 감소. 미니맵 전용 Renderer로 Camera Render 50% 이상 절감.
